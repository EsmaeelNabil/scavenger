## Scavenger
Collect and analyze websites code and sources for secrets and juicy information.

### CheckList
- [-] Collection

### Usage

#### Scavenger-extension
- Go to chrome/arc or any chromuim based browsers extensions [tab](chrome://extensions/)
- Load unpacked
- Choose the `scavenger-extension` folder

#### Scavenger-processor
- run `go run main.go`
